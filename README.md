# SoundWire
SoundWire — это кроссплатформенный мессенджер, объединяющий социальное взаимодействие и музыкальный стриминг. Приложение позволяет пользователям общаться в чатах, обмениваться треками, создавать совместные плейлисты и проводить голосовые комнаты с синхронным прослушиванием. В основе бизнес-процесса лежит интеграция стримингового контента в коммуникационные функции.

Авторы - Дмитрий Поликарпов, Илья Егоров, Александр Крутицкий.
